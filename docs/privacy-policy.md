# Privacy Policy — FB Enhancer

**Last updated: July 8, 2026**

---

## Introduction

FB Enhancer ("the Extension") is a browser extension for Google Chrome and Microsoft Edge that helps you clean up your Facebook feed by hiding unwanted content such as sponsored posts, stories, reels, suggested content, and the right sidebar.

This Privacy Policy explains what information the Extension does and does not collect, and how it operates.

---

## Information We Collect

**We collect no personal information whatsoever.**

FB Enhancer does not collect, store, transmit, or share:

- Your name, email address, or any identifying information
- Your Facebook account details or profile data
- Your browsing history or visited URLs
- The content of any web pages you visit
- Your IP address or device identifiers
- Cookies or session data
- Usage statistics or analytics

---

## How the Extension Works

FB Enhancer operates entirely on your device. It injects a content script into Facebook pages that:

1. Scans the page DOM for elements matching known patterns (sponsored posts, stories, reels, etc.)
2. Hides matching elements using CSS (`display: none`)
3. Monitors for new elements added by Facebook's dynamic feed using the browser's MutationObserver API

All of this processing happens locally in your browser. No page content, element data, or user activity is ever read for any purpose other than deciding which elements to hide visually.

---

## Data Storage

The only data stored by the Extension is your feature toggle preferences — which of the six filters you have enabled or disabled.

- **What is stored:** A single JSON object, e.g. `{ "hide-stories": true, "hide-reels": false, ... }`
- **Where it is stored:** Your browser's built-in `chrome.storage.sync` API
- **Who can access it:** Only the Extension itself. No other website, service, or third party can access this data.
- **Is it sent anywhere:** `chrome.storage.sync` may sync this setting object across your own signed-in browser instances (e.g., between your laptop and desktop if you use Chrome Sync or Edge Sync). This sync is handled entirely by your browser and Google/Microsoft — FB Enhancer has no involvement in or visibility into this sync process.
- **When is it deleted:** You can remove it at any time by uninstalling the Extension or clearing extension storage in your browser settings.

## Third-party Services

FB Enhancer uses **no third-party services** of any kind.

There are no:

- Analytics providers (Google Analytics, Mixpanel, Segment, etc.)
- Error tracking services (Sentry, Bugsnag, etc.)
- Advertising networks
- Remote configuration services
- CDN-hosted scripts
- External APIs

---

## Remote Code

FB Enhancer does **not** execute any remote code.

All JavaScript that runs as part of this Extension is bundled and included in the extension package at install time. No scripts are fetched from external URLs at runtime. This is verifiable by inspecting the extension's source code and the compiled output in the `dist/` directory.

---

## Permissions

The Extension requests the following browser permissions:

| Permission | Purpose |
|---|---|
| `storage` | Save your toggle preferences locally so they persist between sessions |
| `host_permissions: facebook.com` | Required to inject the content script that hides unwanted elements on Facebook pages |

Neither permission is used to collect, read, or transmit personal data. See [`docs/permissions.md`](./permissions.md) for a full explanation of each permission.

---

## Security

Because FB Enhancer operates entirely on your device with no external communication:

- There is no server that could be breached to expose your data
- There is no database containing your information
- There is no network traffic generated by the Extension that could be intercepted
